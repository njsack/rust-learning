# headr
```
headr 0.1.0

USAGE:
    headr [OPTIONS] [FILES]...

ARGS:
    <FILES>...    Input file(s) [default: -]

OPTIONS:
    -c, --bytes <BYTES>    Number of bytes to read
    -h, --help             Print help information
    -n, --lines <LINES>    Number of lines to read [default: 10]
    -V, --version          Print version information
```
