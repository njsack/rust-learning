# catr
```
catr 0.1.0

USAGE:
    catr [OPTIONS] [FILE]...

ARGS:
    <FILE>...    Input file(s) [default: -]

OPTIONS:
    -b, --number-nonblank-lines    Number nonblank lines?
    -h, --help                     Print help information
    -n, --number-lines             Number lines?
    -V, --version                  Print version information
```
